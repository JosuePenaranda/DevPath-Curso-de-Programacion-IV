¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Permite organizar el contenido mediante etiquetas que indican al navegador cómo mostrar textos, imágenes, enlaces y otros elementos.

Estructura básica de un documento HTML

Todo documento HTML tiene una estructura básica que organiza el contenido de la página. Incluye las etiquetas principales que indican al navegador dónde empieza el documento, la información del sitio y el contenido visible.

Etiquetas principales
HTMLEtiqueta raíz que indica el inicio y fin del documento.
HeadContiene metadatos, título y enlaces a estilos.
BodyTodo el contenido visible de la página.
TitleTítulo que aparece en la pestaña del navegador.
MetaInformación adicional como codificación y descripción.
Encabezados (h1 – h6)

Los encabezados se utilizan para organizar el contenido en títulos y subtítulos. Van desde h1 (más importante) hasta h6 (menos importante).

Párrafos (p)

La etiqueta <p> se utiliza para escribir y organizar bloques de texto dentro de una página web.

Enlaces (a)

La etiqueta <a> permite crear enlaces que llevan a otras páginas web, documentos o secciones dentro de la misma página.

Imágenes (img)

La etiqueta <img> se usa para insertar imágenes en una página web mediante la dirección o ruta del archivo de imagen.

Listas
  • olLista ordenada — elementos numerados
  • ulLista no ordenada — elementos con viñetas
Tablas

Las tablas permiten organizar información en filas y columnas, facilitando la presentación estructurada de datos.

Formularios

Los formularios permiten a los usuarios ingresar información en una página web, como datos personales, búsquedas o registros.

Inputs y tipos de input

Los inputs son campos dentro de los formularios que permiten ingresar diferentes tipos de datos, como texto, contraseñas, correos electrónicos, fechas o números.

Etiquetas semánticas
headerParte superior de la página, contiene título o logo.
navAgrupa enlaces de navegación.
sectionDefine una sección temática del contenido.
articleContenido independiente como un artículo o noticia.
footerParte final de la página con info de contacto.
