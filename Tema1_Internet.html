TEMA 1: Fundamentos de Internet y la Web

Internet, Web, Modelo OSI, HTTP, HTTPS y más

¿Qué es el Internet?

Internet es una red mundial de computadoras conectadas entre sí que permite compartir información y comunicarse mediante distintos servicios y protocolos.

¿Qué es la World Wide Web?

La World Wide Web (Web) es un sistema de páginas y recursos enlazados que se acceden a través de Internet usando navegadores web.

Diferencia entre Internet y Web

Internet es la infraestructura de red global, mientras que la Web es un servicio que funciona sobre Internet para acceder a páginas web.

Modelo OSI

El Modelo OSI (Open Systems Interconnection) es un modelo conceptual que explica cómo se comunican los sistemas en una red mediante diferentes capas.

Capas del modelo OSI
  • Física
  • Enlace de datos
  • Red
  • Transporte
  • Sesión
  • Presentación
  • Aplicación
Dirección IP

Una dirección IP es un identificador único asignado a cada dispositivo conectado a una red para poder comunicarse con otros dispositivos.

DNS (Domain Name System)

El DNS es un sistema que traduce nombres de dominio (como google.com) en direcciones IP para que los navegadores puedan localizar los servidores.

Servidores y clientes

Un cliente es el dispositivo que solicita información (como un navegador), mientras que un servidor es el equipo que responde y entrega los recursos solicitados.

Navegadores web

Los navegadores web son programas que permiten acceder y visualizar páginas web, interpretando el código de los sitios. Ejemplos: Chrome, Firefox o Edge.

Funcionamiento de una petición web

Cuando un usuario escribe una URL, el navegador solicita la página a un servidor mediante HTTP o HTTPS. El servidor procesa la solicitud y envía la respuesta con los datos del sitio web.

Protocolo HTTP

HTTP (HyperText Transfer Protocol) es el protocolo que permite la transferencia de información entre el navegador y el servidor web.

Protocolo HTTPS

HTTPS es una versión segura de HTTP que cifra la información transmitida entre el cliente y el servidor.

Diferencia entre HTTP y HTTPS

HTTP transmite datos sin cifrado, mientras que HTTPS protege la información mediante cifrado para mayor seguridad.

SSL y TLS

SSL y TLS son protocolos de seguridad que cifran la comunicación entre el navegador y el servidor para proteger la información.

Métodos HTTP
  • GETObtener información
  • POSTEnviar datos al servidor
  • PUTActualizar datos
  • DELETEEliminar datos
Códigos de estado HTTP
  • 200Solicitud exitosa
  • 404Recurso no encontrado
