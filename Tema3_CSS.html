¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para dar estilo y diseño a las páginas web. Permite controlar colores, tamaños, posiciones y la apariencia visual de los elementos creados con HTML.

Formas de aplicar CSS
  • 1En línea — atributo style en la etiqueta
  • 2Interno — etiqueta <style> en el head
  • 3Externo — archivo .css separado (recomendado)
Selectores principales
  • etiquetaAplica a todas las etiquetas del mismo tipo
  • .claseAplica a elementos con esa clase
  • #idAplica a un elemento único
  • descendenteElementos dentro de otros
  • :hoverPseudo-clases para estados
  • ::beforePseudo-elementos para partes
Propiedades principales
  • color — color del texto
  • background — fondo del elemento
  • font-family — tipo de letra
  • font-size — tamaño del texto
  • width / height — ancho y alto
  • margin / padding — espaciado
  • border — bordes
Modelo de caja (Box Model)

El modelo de caja describe cómo se estructura un elemento en CSS. Incluye el contenido, el padding, el borde y el margen, que juntos determinan el tamaño y el espacio que ocupa el elemento.

Display

La propiedad display controla cómo se muestra un elemento en la página, por ejemplo como bloque, línea o contenedor flexible.

Position

La propiedad position permite controlar la ubicación de un elemento dentro de la página, utilizando valores como static, relative, absolute o fixed.

Flexbox

Flexbox es un sistema de diseño que facilita la alineación y distribución de elementos dentro de un contenedor, permitiendo crear diseños flexibles y responsivos.

Buenas prácticas de estilos

Las buenas prácticas en CSS incluyen mantener el código organizado, usar archivos externos, reutilizar clases y evitar estilos innecesarios para mejorar la claridad y mantenimiento del diseño.
